# CSNETWK-FileTransferMP

## Instructions

### Starting a server:

``` bash
py Server.py <ip address> <port>
```



### Starting a client:

#### CLI
```bash
py Client.py
```

#### GUI
```bash
py View.py
```

If there is a need to clear /Server/Files , make sure to also clear the list of files in FileList.json (for GUI implementation)
